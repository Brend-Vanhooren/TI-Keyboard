This program is designed for the TI-84 plus CE-T Python and thus only tested on it. This was made for personal use.
**Still under development!**

How to use:
- Compiling:
    - Use CEdev from the CE C/C++ Toolchain: https://ce-programming.github.io/toolchain/static/getting-started.html
- Installation
    - Use TI Connect to put the Keyboard.8xp file on the TI-84 (found in the bin folder)
    - The clibs must also be on the TI-84: https://github.com/CE-Programming/libraries/releases/tag/v13.0
    - To launch the file u need somthing like Cesium: https://www.youtube.com/watch?v=_e8pgw9d7S4
